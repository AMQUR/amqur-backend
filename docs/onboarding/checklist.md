# Onboarding checklist

- [ ] DealerGroup created/linked
- [ ] Tenant + location slugs reserved
- [ ] `allowedOrigins` exact HTTPS origins
- [ ] Branding `publicConfig` set (assistant name, colors, logo, disclaimer)
- [ ] Consent text + retention days
- [ ] Escalation recipients
- [ ] Inventory feed URL authorized + allowlisted (or inventory flag left false)
- [ ] CRM webhook or Tekion path decided
- [ ] Admin user created; password rotated
- [ ] Feature flags fail-closed except explicitly approved
- [ ] Widget embed snippet tested on staging host
- [ ] Truthfulness UAT: no fabricated prices/hours/APRs
- [ ] Handoff: durable escalation; no false “notified” claims
